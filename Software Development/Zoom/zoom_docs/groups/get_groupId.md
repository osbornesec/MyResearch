## Retrieve a group

Retrieve a group under your account

> Definition

```shell
GET https://api.zoom.us/v2/groups/{groupId}
```

> Example Request

```shell
$ curl https://api.zoom.us/v2/groups/{groupId}
```

> Example Response

```text
200 OK
```

```json
{
    "id": "string",
    "name": "string",
    "total_members": "integer"
}
```

Path Arguments | &nbsp;
--- | ---
groupId **required** | The group ID
