## Update a user's picture

Update a user's profile picture

> Definition

```shell
PUT https://api.zoom.us/v2/users/{userId}/picture
```

> Example Request

```shell
$ curl https://api.zoom.us/v2/users/{userId}/picture
```

> Example Response

```text
201 Created
```

Path Arguments | &nbsp;
--- | ---
userId **required** | The user ID or email address
