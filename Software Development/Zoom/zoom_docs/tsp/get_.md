## List TSP dial-in numbers

List TSP dial-in numbers under account

> Definition

```shell
GET https://api.zoom.us/v2/tsp
```

> Example Request

```shell
$ curl https://api.zoom.us/v2/tsp
```

> Example Response

```text
200 OK
```

```json
{
    "tsp_provider": "string",
    "dial_in_numbers": [
        {
            "code": "string",
            "number": "string",
            "type": "string"
        }
    ]
}
```
